Run server:
```
mvn spring-boot:run
```
Start server:
```
mvn spring-boot:start
```
Stop server:
```
mvn spring-boot:stop
```
Run tests:
```
mvn test
```
Run individual test:
```
mvn -Dtest=<TestName> test
```
